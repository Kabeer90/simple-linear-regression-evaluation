# Simple Linear Regression – Marketing ROI Analysis

## Project Overview
This repository contains a data science project demonstrating how to use Simple Linear Regression to translate raw marketing data into actionable business insights. Using Python and `statsmodels`, the project analyzes the impact of various advertising channels (TV, Radio, Social Media) on total Sales, identifies the optimal predictive variable, validates OLS regression assumptions, and provides an ROI-based business recommendation.

## Environment Setup
To run this notebook, ensure you have Python installed, then install the required dependencies using pip:
`pip install pandas seaborn statsmodels matplotlib scipy`

## Execution
Open `regression_analysis.ipynb` in Jupyter Notebook or JupyterLab and "Run All" cells to view data cleaning, visualizations, statistical modeling, and final business recommendations.
